
1️⃣ Difference between var, let, and const ? 
var – Function-scoped, can be redeclared and reassigned.

let – Block-scoped, can be reassigned but not redeclared.

const – Block-scoped, cannot be redeclared or reassigned

2️⃣ What is the spread operator (...)?

The spread operator allows you to expand elements of arrays or objects into a new array or object.

3️⃣ Difference between map(), filter(), and forEach() map() Transforms each element and return New array.

filter() Keeps elements meeting a condition and return New array.

forEach() Runs a function for each element and return undefined.

4️⃣ What is an arrow function? A short syntax for writing functions. Does not have its own this, uses the surrounding scope.

5️⃣ What are template literals? Strings enclosed in backticks ` that allow variables and expressions inside ${} and multiline strings.
